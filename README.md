# Photo Portfolio

Dmytro PI photo and art portfolio. The reboot is migrating the project to Vite and rebuilding the experience around a cinematic spatial gallery.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.

Open [http://localhost:5173](http://localhost:5173) to view it in your browser. Vite will choose another port if `5173` is already in use.

### `npm run build`

Builds the app for production to the `dist` folder.

### `npm run preview`

Serves the production build locally for a final smoke test.
